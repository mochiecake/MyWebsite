Capstone Project - Deploying Neural Networks on PLCs for Industrial Predictive Maintenance

Motivation:

Explore the feasibility of deploying neural networks on PLCs for real-time predictive maintenance in industrial environments.

Approach:

Built and trained a neural network using Scikit-learn, TensorFlow, and Keras on synthetic data (due to scarcity of actual industrial machine data).

Outcome:

Successfully deployed ML on a PLC, enabling real-time predictive maintenance with actionable improvement recommendations.


University Project - Early Prediction of Diabetes using Machine Learning

Motivation:

Develop a classification algorithm that accurately predicts diabetes diagnosis to improve patient health outcomes.

Approach:

Tested and compared three algorithms based on their F1 score, accuracy, and AUC that effectively captures relationships in data.

Outcome:

Successfully chosen the most suitable algorithm that has the best balance of F1 score, accuracy, and AUC.

University Project - Tag-along Wheelchair System


Motivation:

Develop a tag-along system to assist elderly individuals with mobility challenges, enabling them to navigate independently.

Approach:

Designed a tag-along wheelchair system that includes a tracking and alert system which features easy attachment and detachment.

Outcome:

Successfully demonstrated the system's ability to pull a user along while avoiding obstacles, enhancing mobility and independence.
